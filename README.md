# Loopback example tests

The project is generated by [LoopBack](http://loopback.io).

## Project features

- Tests
  - [ ] Unit / BIT
  - [x] e2e
- Watcher for TDD
- Coverage

## Commands

Run the tests with coverage analysis:

```bash
npm test
```

Run the test with file watcher:

```bash
npm run test-watch
```
